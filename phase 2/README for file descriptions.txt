1) Airline Reviews Sentiment Analysis: Presentation presented in class on December 07, 2023
Airline_Reviews.csv: Our original dataset sourced from Kaggle.
2) airlines.ipynb: Notebook for data cleaning, transformation and EDA.
3) airlines_type.csv: Transformed dataset. Used for EDA in tableau.
4) Changes Document.docx: The document which shows the changes between phase 1 proposal and phase 2.
5) Department_Statistical Evidence.xlsx: The results of T-test for research questions 4&5.
6) Department_Statistical_evidence.ipynb: The code for the T-test for research questions 4&5.
7) EDA.twb: Tableau workbook for our EDA.
8) Phase 2 Document.docx: Our Phase 2 main document.
9) qns 2&3.xlsx: The T-test results for research questions 2 and 3.
10) qns_2&3.ipynb: The code for T-test for research questions 2 and 3.
11) qns_1,2,3.twb: Tableau workbook for questions 1,2 and 3.
12) qns_4&5.twb: Tableau workbook for questions 4 and 5.
13) sentiment_and_correlation.ipynb: The code for implementing sentiment methodologies and checking correlation between them.
14) statdata.csv: The same "vadertextblob.csv" data, with the department satisfaction levels mapped back to numerical ratings given by the customers. This was used in conducting the T-tests.
15) vadertextblob.csv: The transformed data with the preprocessed review text and sentiment scores from VADER and TextBlob and their categorical sentiment labels.
